# Pharmacy Refill Web Application

A full-stack pharmacy refill system built with:

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- HTML / CSS / JavaScript

## Features
- Online prescription refill form
- Admin dashboard with authentication
- Search & filter refill requests
- Responsive modern UI
- Secure backend API

## Screens
- Home page
- Admin panel

## How to Run
1. Run Spring Boot application
2. Open http://localhost:8080
3. Admin page: /admin.html
